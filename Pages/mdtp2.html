2  SETTING UP AN ENABLING COUNSELLING ENVIRONMENT (SPECIFIC OUTCOME 2)

 

Specific outcome 2: Setting up an enabling counselling environment

SO2AC1

The characteristics of an enabling environment are described with reference to counselling setting

SO2AC2

An enabling environment is created for a specific counselling session.

Learning outcomes

At the end of this specific outcome, learners should be able to

  • Describe the characteristics of an enabling environment for counselling
  • Create an enabling environment for a specific counselling session.
