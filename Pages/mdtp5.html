5  APPLYING A COUNSELLING PROCESS (SPECIFIC OUTCOME 5)

Specific outcome 5: Apply a counselling process in a specific context

SO5:AC1

A counselling process is demonstrated to counsel an individual in an authentic situation using appropriate skills and techniques. 

SO5:AC2

An ethical code of conduct is applied in the counselling process.

SO5:AC3

Respect for diversity of values is demonstrated at various stages of the counselling and problem solving process.

Learning outcomes:

At the end of this unit learners should be able to:

  • Counsel an individual in an authentic situation using appropriate skills and techniques.
  • Apply ethical code of conduct in the counselling process.
  • Demonstrate respect for diversity of values at various stages of the counselling and problem solving process.
